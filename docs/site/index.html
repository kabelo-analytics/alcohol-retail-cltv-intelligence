Methodology

Built to be explainable, not just accurate.

The workflow was designed around commercial usability. Every modeling step needed to land cleanly in a sales, CRM, or leadership conversation.

01
Data generation and preparation
Synthetic transactions were used to mirror seasonality, outlet-type behavior, and provincial variation.
02
Feature engineering
Recency, frequency, monetary value, tickets per month, and average order value were derived at outlet level.
03
CLTV modeling and segmentation
Scores were translated into three commercially useful segments: Cash Cows, Occasional Big Spenders, and Core / Mid-Tier.
04
Delivery
The output was turned into a shareable project narrative and dashboard-oriented visual package for weekly planning.
